Gestione sconti automatici per WooCommerce, con supporto ai coupon automatici, alle regole VIP e agli ordini manuali da backoffice.

== Description ==

WSVD gestisce sconti automatici per WooCommerce con una struttura modulare pronta per l'uso in produzione.

Il plugin aiuta il negozio a velocizzare sia le promozioni automatiche sul frontend sia la gestione operativa del backoffice, mantenendo le regole centralizzate direttamente nei coupon WooCommerce.

Le funzioni principali sono:

* coupon automatici basati su restrizioni email
* sconti VIP configurabili per cliente o globali
* pulsante admin per applicare automaticamente i coupon cliente durante la creazione o modifica ordine
* riepilogo e visualizzazione dettagli dei coupon automatici configurati
* compatibilita con il vecchio bootstrap del template

== Installation ==

1. Carica la cartella del plugin in `wp-content/plugins/`.
2. Attiva il plugin da WordPress.
3. Verifica che WooCommerce sia attivo.
