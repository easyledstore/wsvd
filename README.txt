Gestione sconti automatici per WooCommerce, con supporto ai coupon automatici e alle regole VIP.

== Description ==

WSVD gestisce sconti automatici per WooCommerce con una struttura modulare pronta per l'uso in produzione.

Le funzioni principali sono:

* coupon automatici basati su restrizioni email
* sconti VIP configurabili per cliente o globali
* compatibilita con il vecchio bootstrap del template

== Installation ==

1. Carica la cartella del plugin in `wp-content/plugins/`.
2. Attiva il plugin da WordPress.
3. Verifica che WooCommerce sia attivo.
